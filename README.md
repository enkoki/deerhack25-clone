# Deerhack25 Clone

This project is a **Deerhack25 clone**, created while **learning React and Tailwind CSS**.  The offical Deerhack 25 website: https://deerhack.deerwalk.edu.np/

---

## Tools & Languages Used

- React
- Tailwind CSS
- Vite 
- Node.js & npm

---

## Dependencies

Before running this project, make sure you have the following installed:

* **Node.js** (includes npm)

  * Download from: [https://nodejs.org/](https://nodejs.org/)

You can verify installation by running:

```bash
node -v
npm -v
```

---

## Setup & Run Locally

Follow these steps to run the project on your desktop:

1. **Clone the repository**

   ```bash
   git clone https://github.com/enkoki/deerhack25-clone
   cd deerhack25-clone
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open in browser**

   After running the command, open the URL shown in your terminal (usually `http://localhost:5173`).
